Most traders install from MQL5 Market inside MT5 (no file copying). The steps below are only if you received the full AuraFX-Elite folder on your PC.

For customers — bought on MQL5 Market

  1. Open MetaTrader 5 from your broker (free).
  2. Menu Tools → Market (or press Ctrl+M).
  3. Search AuraFX → choose your product → Install.
  4. In Navigator → Indicators → Market, drag the indicator onto your chart.
  5. For the EA: enable Algo Trading, then attach from Expert Advisors.

Advanced — manual install from AuraFX-Elite folder (owners / developers)

Step 1 — Copy files

In MT5: File → Open Data Folder. From your project folder on this PC (Desktop\AuraFX-Elite\MQL5\), copy:

  • Indicators\ → MT5 MQL5\Indicators\
  • Experts\ → MT5 MQL5\Experts\
  • Include\ → MT5 MQL5\Include\
  • Presets\ → MT5 MQL5\Presets\ (optional)

Step 2 — Compile

Press F4 (MetaEditor) → open each .mq5 → F7 Compile. Expect 0 errors.

Step 3 — Attach

Navigator → Indicators → Custom → drag AuraFX_Elite_Signals or AuraFX_Pro_Assistant onto chart.
