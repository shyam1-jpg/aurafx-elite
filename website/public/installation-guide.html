Installation guide

Educational software only — not financial advice

Option A — MQL5 Market (recommended)

  1. Download MetaTrader 5 from your broker (free).
  2. Open MT5 → Tools → Market (Ctrl+M).
  3. Search AuraFX → select product → Install.
  4. Navigator → Indicators → Market → drag indicator onto chart.
  5. For Expert Advisors: enable Algo Trading toolbar button, then attach EA from Navigator.
  6. Allow algorithmic trading in Tools → Options → Expert Advisors if prompted.
