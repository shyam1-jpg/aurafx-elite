Live institutional preview (owner only)

Public visitors still see demo data. Only you can turn on live FX/crypto snapshots while you finish COT, whale, and ETF APIs.

Server must run with INSTITUTIONAL_MODE=preview. Default is demo (live API blocked).

Step 1 — Start server in preview mode

Windows CMD (in website folder):

set INSTITUTIONAL_MODE=preview
set AURAFX_OWNER_KEY=aurafx-owner
node simple-server.js

Or double-click START-LIVE-PREVIEW.bat in the project folder.

Step 2 — Save owner key in this browser

Open dashboard with live preview

When ready for everyone: set INSTITUTIONAL_MODE=public on the server (later). Until then, do not link this page from the public home page.
