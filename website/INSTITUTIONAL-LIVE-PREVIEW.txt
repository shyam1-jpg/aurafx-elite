AURAFX — LIVE INSTITUTIONAL (NOT PUBLIC YET)
==========================================

THREE MODES (environment variable INSTITUTIONAL_MODE):

  demo     — DEFAULT. Public + owner see demo COT/whale text. /api/institutional-live returns 403.
  preview  — Live FX/crypto for OWNER ONLY (X-Owner-Key). Public still sees demo.
  public   — (Future) Live for everyone when you are ready to launch.

HOW TO USE PREVIEW MODE (you only)
----------------------------------
1. Run START-LIVE-PREVIEW.bat  (sets INSTITUTIONAL_MODE=preview)
2. Open http://127.0.0.1:3847/owner-preview.html
3. Enter owner key (default: aurafx-owner)
4. Click "Save key & test live API"
5. Open dashboard — green banner "OWNER LIVE PREVIEW"
