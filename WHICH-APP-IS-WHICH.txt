HOW TO TEST THE BROWSER PAGE
----------------------------
Open: pro-dashboard\index.html

You should see gold/black theme and a status bar at top.

Click these to test each feature:
  [Use demo data]     - refreshes demo numbers
  [Test crash alert]  - red warning banner
  [Test high score]   - score 85, checklist PASS

Journal section:
  - Enter symbol, entry, exit, click Save trade
  - Trade should appear in list below

Screenshot section:
  - Choose any chart image file
  - Green/red zones should appear on canvas

Status bar says:
  DEMO MODE = page works; optional server not running
  LIVE API  = news server connected (TEST-START.bat)
